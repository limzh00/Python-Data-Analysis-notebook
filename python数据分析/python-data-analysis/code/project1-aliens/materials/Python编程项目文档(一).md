## Section 1.序言

今天我们进入面向对象项目的编程，目的是实践，无赘述。

在外星人入侵中，我们要实现的功能有：

- 飞船可以左右移动，并用空格键射击；
- 外星人群从左移动到屏幕右端然后下移一定距离，接着往反方向移动；
- 飞船射杀外星人，也就是子弹与外星人碰撞时，子弹与外星人消失，外星人消灭完后又新创建一群；
- 当外星人撞到飞船，玩家就损失一艘飞船，满3次游戏结束；
- 记录得分，最高得分，等级，余下的飞船数。
